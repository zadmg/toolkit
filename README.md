# 自动化工具

> CLI工具、自动化脚本、AI Agent相关工具

## 项目列表

### 1. cli-anything 【Fork】

- **项目地址**: https://github.com/HKUDS/CLI-Anything
- **原作者**: [HKUDS](https://github.com/HKUDS)
- **说明**: CLI自动化工具集 - 让AI Agent通过命令行控制84种软件

#### 功能特点

- 支持控制84种桌面软件和Web服务
- 覆盖领域：3D建模/AI图像/音频/自动化/通讯/办公/视频剪辑/浏览器等
- AI Agent原生设计
- CLI-Hub管理工具
- 社区贡献支持

#### 支持的软件类别

- 3D建模软件
- AI图像处理
- 音频编辑
- 自动化工具
- 通讯软件
- 办公软件
- 视频剪辑
- 浏览器控制

#### 安装使用

```bash
pip install cli-anything-hub
cli-hub install <name>
```

#### 特性亮点

- 让传统软件具备Agent能力
- 简单的CLI接口
- 支持预览和实时预览
- 轨迹循环功能
