## 项目列表

### 1. pentagi 【Fork】

- **项目地址**: https://github.com/vxcontrol/pentagi
- **原作者**: [vxcontrol](https://github.com/vxcontrol)
- **说明**: AI驱动渗透测试自动化平台

#### 功能特点

- 内置20+安全工具（nmap/metasploit/sqlmap等）
- 支持全自主模式和辅助模式
- 多代理协作架构
- 支持10+ LLM提供商
- 自动化渗透测试流程

#### 适用场景

- 企业安全评估
- 渗透测试
- 漏洞扫描
- 安全培训

### 2. help_tool 【原创】

- **项目地址**: https://github.com/zadmg/help_tool
- **作者**: [zadmg](https://github.com/zadmg)
- **说明**: 推理算法助手 - 加密算法识别与解密工具

#### 功能特点

- 自动识别加密算法类型（DES/AES/RSA/SM3/SM4等）
- 尝试解密并返回明文
- 支持多种加密模式（CBC/ECB等）
- 支持真机进程内存分析
- 简洁易用的GUI界面

#### 适用场景

- 安全研究
- 数据恢复
- 加密分析
- 软件逆向

#### 支持的算法

- MD5
- SHA系列
- AES/DES/3DES
- SM3/SM4
- RSA
